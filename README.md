# 🚀 Advanced Automated EDA & Data Preprocessing Tool

An intelligent, end-to-end **Exploratory Data Analysis (EDA) and Data Preprocessing platform** built using Streamlit. This tool automates data cleaning, transformation, feature engineering, and model experimentation — reducing manual effort and improving efficiency.

---

## 💡 Project Overview

In real-world projects, poor data quality (missing values, noise, inconsistencies) significantly impacts model performance.

This project introduces an **Intelligent Refinery Approach** to automate:

* Data cleaning
* Preprocessing
* Feature engineering
* Model experimentation

It provides a **user-friendly interface** where users can upload datasets and perform advanced data operations without writing code.

## ▶️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/Jhanu06/Exploratory-Data-analysis-Tool.git
cd Exploratory-Data-analysis-Tool
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the application

```
streamlit run streamlit_app.py
```
